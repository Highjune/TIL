## 이클립스 단축키
- 해당하는 메서드를 호출하는 메서드 찾기
```
ctrl + shift + G
```
- 파일 찾기
```
ctrl + shift + R
```
- 변수 일괄 변경
```
alt + shift + R
```
- 현재 창 크게
```
ctrl + M
```
- 특정 영역만 선택
```
alt + shift + A
```
- 이크립스 화면 초기화 
```
window - Perspective - Rest Perspective
```
- 클래스 내 메서드 찾기
```
ctrl + O
```
- 이전 페이지로 돌아가기
```
art + (좌우 화살표, ↔)
```
