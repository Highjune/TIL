# 인텔리J

## 단축키
- build.gradle안에서 compile메소드의 자동완성
```
ctrl + space
```
- action 
    - 윈도우
    ```
    ctrl + shift + a
    ```
    - 맥
    ```
    command + shift + a
    ```

- 프로젝트에서의 새 파일생성(프로젝트 클릭후)
```
art + insert
```

- 깃 커밋창 열기(물론 깃 연동 후)
    - 윈도우
    ```
    ctrl + k
    ```
    - 맥
    ```
    command + k
    ```

- 깃 push 
    - 윈도우
    ```
    ctrl + shift + k
    ```
    - 맥
    ```
    command + shift + k
    ```
- 패키지 가져오기
    - 윈도우/리눅스
    ```
    alt + enter
    ```
    - 맥
    ```
    option + enter
    ```

- 파일 이름 변경
    - 윈도우
    ```
    shift + f6
    ```
- 라인 단위 복사
    - 윈도우/리눅스
    ```
    ctrl + d
    ```

- 라인 단위 옮기기
    - 윈도우/리눅스
    ```
    alt + shift +  ↑↓
    ```
- 라인 단위 삭제
    - 윈도우/리눅스
    ```
    ctrl + y
    ```
- 해당하는 메서드로 이동
    - 윈도우/리눅스
    ```
    ctrl + b
    ```
